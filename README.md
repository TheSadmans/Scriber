# Scriber — Digital Marketing Agency

Bangladesh's creative digital marketing agency website. Designed for landing pages, motion posters, logos, websites & web apps.

## 📁 Project Structure

```
scriber-agency/
├── index.html      # Main HTML file (structure & content)
├── style.css       # All styles (dark theme, animations, responsive)
├── script.js       # All JavaScript (cursor, orders, admin panel)
└── README.md       # This file
```

## 🚀 Features

- Custom animated cursor
- Page loader animation
- Responsive navigation with mobile menu
- Hero section with animated orbs
- Services grid
- Portfolio section with filter buttons
- Process steps
- Testimonials
- Order form with localStorage
- Admin panel (password: `scriber2025` — **change this!**)
- Footer

## 🌐 Live Hosting (GitHub Pages)

1. Go to **Settings → Pages**
2. Set Branch to `main`, folder to `/ (root)`
3. Save — your site will be live at:  
   `https://YOUR-USERNAME.github.io/scriber-agency/`
